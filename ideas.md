# Ideas de Diseño para Ululato

## Filosofía Seleccionada: **Neo-Brutalism Dark Edition**

### Movimiento de Diseño
Neo-Brutalismo adaptado al modo oscuro con elementos de diseño premium y sofisticación corporativa. Combina la audacia del brutalismo moderno con la elegancia de interfaces de alta gama.

### Principios Fundamentales
1. **Contraste Dramático**: Uso intensivo de fondos ultra-oscuros (#09090b, #18181b) con elementos brillantes en dorado (#FFD700) y azul marino (#003366) para crear jerarquía visual clara
2. **Tipografía Expresiva**: Combinación de fuentes display bold para títulos con sans-serif limpia para cuerpo, creando ritmo visual
3. **Bordes y Sombras Definidas**: Uso de borders gruesos en colores de marca, sombras profundas con glow effects en elementos interactivos
4. **Espaciado Generoso**: Whitespace (o "darkspace") abundante para respiración visual y enfoque en contenido premium

### Filosofía de Color
- **Fondo Base**: Negro profundo (#09090b) como lienzo principal, transmitiendo sofisticación y profesionalismo
- **Dorado Ululato (#FFD700)**: Color de acción y éxito - botones CTA, progress bars, precios, badges de "Bestseller", elementos de logro
- **Azul Marino (#003366)**: Confianza y estructura - headers, footers, secciones hero, elementos de navegación secundaria
- **Rojo Crimson (#DC143C)**: Urgencia y atención - descuentos, notificaciones importantes, badges de "Hot"
- **Grises Oscuros**: Variaciones de #18181b a #27272a para cards, modales, y elementos elevados

### Paradigma de Layout
- **Grid Asimétrico con Énfasis**: Layouts que rompen la simetría tradicional con elementos destacados (hero sections con diagonal cuts, cards con diferentes heights)
- **Sidebar Contextual**: Navegación lateral sticky para dashboards y curso player
- **Carruseles Cinemáticos**: Hero carousels con transiciones suaves y overlays de gradiente
- **Cards con Elevación**: Sistema de elevación de 3 niveles usando sombras y borders para crear profundidad

### Elementos Distintivos
1. **Glow Effects**: Elementos interactivos con subtle glow en hover usando box-shadow con colores de marca
2. **Progress Indicators Premium**: Barras de progreso con gradientes dorados y animaciones fluidas
3. **Badge System**: Sistema de badges con borders gruesos, backgrounds semi-transparentes y iconografía clara

### Filosofía de Interacción
- **Feedback Inmediato**: Todas las interacciones tienen respuesta visual instantánea (hover states, active states)
- **Transiciones Suaves**: Uso de ease-out curves para transiciones naturales (duration: 200-300ms)
- **Micro-animaciones**: Elementos como badges, notifications y progress bars tienen subtle animations al aparecer
- **States Claros**: Estados de hover, active, disabled y selected son visualmente distintos

### Animación
- **Entrada de Elementos**: Fade-in combinado con slide-up (translate-y) para cards y sections
- **Hover Effects**: Scale subtle (1.02-1.05) con glow effect en botones y cards
- **Progress Animations**: Barras de progreso se animan con transition en width
- **Loading States**: Skeleton screens con shimmer effect en color dorado

### Sistema Tipográfico
- **Display/Headings**: **Space Grotesk** (Bold 700, SemiBold 600) - Geométrica, moderna, con personalidad
- **Body/UI**: **Inter** (Regular 400, Medium 500) - Legible, profesional, optimizada para pantallas
- **Jerarquía**:
  - H1: 48px/56px (3rem/3.5rem) - Space Grotesk Bold
  - H2: 36px/44px (2.25rem/2.75rem) - Space Grotesk SemiBold
  - H3: 24px/32px (1.5rem/2rem) - Space Grotesk SemiBold
  - Body: 16px/24px (1rem/1.5rem) - Inter Regular
  - Small: 14px/20px (0.875rem/1.25rem) - Inter Regular
